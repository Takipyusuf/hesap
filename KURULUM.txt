BORÇ TAKİP — KURULUM REHBERİ
============================

1) FIREBASE (ücretsiz)
   - https://console.firebase.google.com → Yeni proje oluştur
   - Authentication → Sign-in method → E-posta/Parola → Etkinleştir
   - Firestore Database → Veritabanı oluştur

2) firebase-config.js DOSYASINI DOLDUR
   - Firebase → Proje ayarları → Uygulamanız → Web (</>) ekle
   - Config bilgilerini firebase-config.js içine yapıştır
   - ADMIN_EMAIL satırına KENDİ e-postanıı yaz (yönetici olacaksın)

3) FIRESTORE KURALLARI
   - Firebase → Firestore → Kurallar
   - firestore.rules dosyasını aç
   - İçindeki "yonetici@email.com" kısmını ADMIN_EMAIL ile AYNI yap
   - Kuralları yapıştır → Yayınla

4) YÖNETİCİ HESABI OLUŞTUR
   - index.html sayfasını tarayıcıda aç
   - Kayıt Ol → ADMIN_EMAIL ile kayıt ol (şifreni belirle)
   - admin.html → aynı e-posta ve şifre ile giriş yap

5) GITHUB PAGES İLE YAYINLA (önerilen — kalıcı link)
   - Detaylı adımlar: GITHUB-KURULUM.txt dosyasına bakın
   - Özet: github.com → yeni repo → dosyaları yükle → Settings → Pages → main/root
   - Link: https://KULLANICI.github.io/hesap/
   - Firebase → Authorized domains → github.io adresini ekle

   Alternatif: https://app.netlify.com/drop (aynı dosyaları sürükle)

KULLANIM
--------
- Kullanıcılar: index.html → Kayıt ol / Giriş yap
- Sen (yönetici): admin.html → Tüm kullanıcılar, borçlar, ödemeler, işlem geçmişi

NOT: Firebase ücretsiz planda günlük limit vardır; kişisel kullanım için yeterlidir.
